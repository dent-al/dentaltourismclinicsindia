# Dental Tourism Backend

This is a Node.js Express backend for a dental tourism app. It uses MongoDB, JWT authentication, and follows best practices for REST APIs.

## Features
- User registration & login
- Clinic, dentist, appointment, and payment endpoints
- JWT authentication
- MongoDB models

## Getting Started
1. Install dependencies:
   ```
   npm install
   ```
2. Set up your `.env` file (see `.env` for example).
3. Start the server:
   ```
   node server.js
   ```

## Folder Structure
- `routes/` - API route definitions
- `controllers/` - Route logic
- `models/` - Mongoose models
- `config/` - DB and JWT config
